# 🚀 Backend ReactGram

## 📌 Descrição
Este projeto é um backend desenvolvido com **Node.js** e **Express**, utilizando **MongoDB** com o **Mongoose**.

## 🛠️ Tecnologias utilizadas
- Node.js
- Express
- MongoDB + Mongoose
- JSON Web Token (JWT)
- Express Validator
- Multer
